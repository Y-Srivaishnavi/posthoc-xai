## Introduction
Interpretability is the ability to determine how a model has reached a particular outcome. Explainability is the ability to present a model's reasoning in human-understandable form (*see also: <a href="https://docs.aws.amazon.com/whitepapers/latest/model-explainability-aws-ai-ml/interpretability-versus-explainability.html"> Explainability vs Interpretability </a>*). 

A machine learning model may be made interpretable with the following strategies:
<img src="https://github.com/Y-Srivaishnavi/posthoc-xai/blob/main/slides/Slide3.PNG" width="750" />

While pre-model techniques relate to understanding of data, post-hoc methods of XAI are based on understanding a model's decision making process.
